# Description
Windows10で動作テストしました。
インストールの必要はありません。zipから取り出したexeを(ダブルクリックすれば)実行できます。
zipから直接起動すると設定ファイルを作成できないため、実行できないかもしれません。解凍して取り出してから実行してください。

# Environment
.NET Framework4.8が必要です。
